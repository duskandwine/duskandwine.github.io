环境安装

MacM1下安装miniconda及pytorch：

  • 需要通过安装conda来激活虚拟环境。

  • M1版本的miniconda可以通过官网下载bash文件Miniconda — Conda documentation(其他系统找到自己的版本即可)，然后通过bash Miniconda3 macOS Apple M1 ARM 64-bit bash来进行安装。

  • 之后一直回车和yes就可以完成安装，重启终端发现命令行前出现base则安装成功。

  • 在终端输入命令进行虚拟环境的创建并指定python版本：conda create --name my_env_name python=3.9 。

  • 激活刚刚创建的虚拟环境：conda activate my_env_name ，退出虚拟环境命令为conda deactivate。

  • 进入虚拟环境后首先升级一下pip：python3 -m pip install --upgrade pip。

  • 然后安装M1版本的pytorch，通过搜索发现pytorch已经支持M1芯片了，不过应该是不支持它自研的gpu，通过命令直接安装：conda install -c pytorch pytorch torchvision。

  • 之后利用pip3安装jupyter以及d2l：pip3 install jupyter d2l。

  • 下载李沐老师的教材：《动手学深度学习》，这里可以下载jupyter记事本版本。

  • 然后再安装李沐老师提到一个jupyter插件(幻灯片插件，安不安装都行)：pip3 install rise。 大功告成！

数据操作+数据预处理

  • N维数组是机器学习和神经网络的主要数据结构。

  • 张量表示一个数值组成的数组，这个数组可能有多个维度。

    • 张量用来存储高阶数组，但本质上标量，向量，矩阵都可以看作是张量的特殊形式。

    • 关于张量的详细介绍：什么是张量？ - 知乎 (zhihu.com)

  • 即使形状不同，我们仍然可以通过调用广播机制(broadcasting mechanism)(最容易出错的地方)来执行按元素操作。

    1

3
4

    a = torch.arange(3).reshape((3, 1))
b = torch.arange(2).reshape((1, 2))
a, b 
a + b #这里进行相加的时候，a和b的形状是不一样的，这里会有一个特殊的机制来进行相加，首先这里ab的维度都是2，这里a会把第二维度复制两次与b的第二维度相同，而b会把第一维度复制三次与a相同，然后再进行相加，相加结果是一个3x2的矩阵，这就是广播机制。有些情况下我们可能不需要广播机制，但它进行了。

os.makedirs(os.path.join('..', 'data'), exist_ok=True) #创建了一个文件data
data_file = os.path.join('..', 'data', 'house_tiny.csv') #创建了一个.csv文件，

    import pandas as pd #pandas也是进行数据处理中常用的模块

data = pd.read_csv(data_file) #读取刚刚创建的数据集
print(data)

  • 这里会显示有缺失的数据，对于数据科学家来说，最重要的就是如何处理确实的数据，换言之整个机器学习就是处理缺失的数据，要预测未来缺失的数据。为了处理缺失的数据，典型的方法包括差值和删除，这里我们用到差值。

    1
2
3

    inputs, outputs = data.iloc[:, 0:2], data.iloc[:, 2] #inputs：把第0和第1列还有所有行拿出来 outputs：最后一列拿出来
inputs = inputs.fillna(inputs.mean()) #把na区域填充为不是na部分的值得平均值，也就是3
print(inputs)

  • 对于inputs中的类别值或离散值，将“NaN”视为一个类别。 get_dummies的具体用法特征提取之pd.get_dummies()。

    inputs = pd.get_dummies(inputs, dummy_na=True)
print(inputs)
